# YAWIEL
[![Build Status](https://travis-ci.org/yawiel/yawiel.svg?branch=master)](https://travis-ci.org/yawiel/yawiel)

Yet Another WIkifier and Entity Linker. This is a collocation extractor for the YAWIEL project.

## Build

Dependencies:
* CMake
* Boost
* OpenMP (optional).

How to build?

### Linux

```shell
$ cd yawiel
$ mkdir build
$ cd build
$ cmake ..
$ make
```
